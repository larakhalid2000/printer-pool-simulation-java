# Printer Pool Simulation

## Description
An Operating Systems academic project implemented in Java that simulates the Producer-Consumer problem.

## Concepts Covered
- Multithreading
- Semaphores
- ReentrantLock
- Producer-Consumer synchronization

## Tools & Technologies
- Java (JDK 21)
- java.util.concurrent
- NetBeans

## My Role
- Implemented synchronization logic
- Analyzed system behavior under different load scenarios
